# L社 全機能リスト（公式サイトより抽出）

## コア配信機能
1. **ステップシナリオ配信** - 友だち追加後の自動ステップ配信、シナリオ分岐
2. **セグメント配信** - タグ・条件による絞り込み一斉配信
3. **リマインダーステップ配信** - 特定日から逆算してカウントダウン配信
4. **キーワード応答** - チャットボット的な自動応答
5. **テンプレート** - 再利用可能なメッセージテンプレート

## ユーザー管理
6. **グルーピングタグ** - 性別/年代/地域/購入商品/興味ジャンルでタグ分け
7. **友だち情報欄管理** - 顧客カルテ（個人情報DB蓄積、LINEと個人情報紐付け）
8. **スコアリング** - URLクリック/購入/問合せ等のアクションにスコア値付与
9. **CSV インポート/エクスポート** - 他DBとの連携

## リッチコンテンツ
10. **セグメントリッチメニュー** - グループ/シナリオ購読者別にリッチメニュー切替
11. **カルーセルパネル** - 選択肢ボタン付きバナーパネル複数配信
12. **回答フォーム** - アンケート/予約/問合せフォーム/LP発行

## 分析・計測
13. **流入経路分析** - HP/ブログアクセス遷移数、購入率、滞在時間、動画視聴時間
14. **クリック計測** - URL短縮＋全体/ユーザーごとクリック数集計
15. **ファネル分析** - コンバージョンファネルの可視化

## 運用・管理
16. **オペレーター機能** - 複数オペレーターでのチャット対応
17. **通知機能** - 友だちのアクションに対する通知
18. **クロスデバイス対応** - PC/スマホ/タブレット管理画面
19. **マルチカレンダー** - カレンダー予約機能
20. **広告連携** - LINE広告との連携

# U社 全機能リスト（公式サイトより抽出）

## U社独自機能（L社にないもの）
1. **ファネル作成** - LP/動画視聴ページ/販売ページ/サンクスページ/アップセルページ一式
2. **メール配信** - ステップメール、リマインドメール、セグメント配信
3. **決済・商品販売** - Stripe/UnivaPay/テレコムクレジット等連携、カード決済
4. **会員サイト作成** - 購入者限定コンテンツ、自動アカウント発行
5. **動画アップロード・配信** - 自前動画ホスティング、途中再生で購入ボタン表示
6. **AIアシスト** - ChatGPTベースのLP構成提案、メッセージ文章作成
7. **イベント・予約** - セミナー/説明会/個別相談、事前決済、リマインド自動送信
8. **パートナー（アフィリエイト）** - 紹介URL発行、成果自動計測、報酬計算
9. **アクション機能** - IF-THEN自動化ルール（Zapier的）、スプシ連携
10. **自動ウェビナー（オートウェビナー）** - 録画セミナーの自動再生スケジュール
11. **ページビルダー** - ノーコードLP/セールスレター作成、テンプレート

## SPECに追加すべき機能（U社由来）
- **ファネルビルダー** - LP→動画→販売→決済→会員サイトの一連のページ生成（LIFF + CF Pages）
- **メール配信** - LINE + メールのデュアルチャネル（SendGrid/SES連携）
- **決済連携** - Stripe/LINE Pay統合、購入→自動タグ→シナリオ発火
- **会員サイト** - 購入者限定ページ（CF Pages + D1認証）
- **オートウェビナー** - 録画→スケジュール配信→申込→リマインド
- **アクション自動化** - IF-THENルールエンジン
- **AIアシスト** - Claude/GPTでメッセージ・LP文案生成
